# Footer Mobile Optimization Instructions

## Objective

The current footer looks overly long and cluttered on mobile devices because all sections are stacked vertically. The goal is to redesign the footer so it uses horizontal space more efficiently, reduces scrolling, and maintains a clean, professional appearance across all screen sizes.

---

# Layout Structure

## Section 1: Brand Information + Studio Information

Instead of stacking these sections one after another, place them side by side on mobile and tablet devices.

### Left Column

* Logo / Favicon
* Gobind Dance Center
* Short description:

  > Preserving heritage through movement. Over three decades of cinematic choreography, cultural education, and artistic excellence.
* Social media icons

### Right Column

* Studio heading
* Gobind Dance Center
* 40-08 76th St, Elmhurst
* New York City, USA
* Phone Number
* Book a Session button

### Desired Mobile Layout

| Brand        | Studio     |
| ------------ | ---------- |
| Logo         | Address    |
| Description  | Phone      |
| Social Icons | CTA Button |

This immediately reduces the footer height and keeps important information visible without excessive scrolling.

---

## Section 2: Navigation + Programs

Place Navigation and Programs in a two-column grid instead of stacking them vertically.

### Navigation

### Programs

---

# Responsive Behavior

## Mobile (<768px)

### Top Area

Use a two-column grid:

* Left: Brand Section
* Right: Studio Section

### Middle Area

Use a two-column grid:

* Left: Navigation
* Right: Programs

### Bottom Area

* Copyright text
* Back-to-top button

---

## Tablet (768px–1024px)

Keep the same two-column layout but increase spacing slightly.

### Recommended Grid

* Brand: 1 column
* Studio: 1 column
* Navigation: 1 column
* Programs: 1 column

Display as a 2×2 grid.

---

## Desktop (>1024px)

Use a traditional 4-column footer layout:

1. Brand
2. Navigation
3. Programs
4. Studio

This provides better balance and takes advantage of available width.

---

# Design Improvements

## Reduce Padding

Current spacing appears too large.

### Recommended

Mobile:

* Footer padding: 24px
* Section gap: 20px

Tablet:

* Footer padding: 32px
* Section gap: 24px

Desktop:

* Footer padding: 48px
* Section gap: 32px

---

## Reduce Text Size

### Mobile

* Heading: 12–13px
* Body Text: 14px
* Links: 14px

### Desktop

* Heading: 14px
* Body Text: 15–16px

---

## Improve Button Size

Current button consumes too much width.

### Mobile

* Height: 40–44px
* Width: Auto
* Padding: 12px 20px

The button should align naturally within the Studio column instead of stretching across the entire footer width.

---

## Social Icons

Keep icons compact.

### Recommended

* Size: 32–36px
* Gap: 8px

Display horizontally below the description.

---

## Visual Hierarchy

### Headings

Use:

* Uppercase
* Small letter spacing
* Accent color

Examples:

* NAVIGATION
* PROGRAMS
* STUDIO

### Content

Use slightly lighter text color for secondary information.

### Divider

Add a subtle divider above the copyright section to separate footer content from the bottom area.

---

# Final Goal

The footer should:

* Use two-column layouts on mobile.
* Minimize unnecessary vertical scrolling.
* Keep Navigation and Programs side by side.
* Keep Brand and Studio sections side by side.
* Maintain excellent readability.
* Look clean, modern, and professional.
* Scale smoothly across mobile, tablet, laptop, and desktop devices.
* Feel visually balanced without appearing crowded.

The overall result should be a compact, elegant, responsive footer that occupies significantly less vertical space while preserving all content and functionality.
